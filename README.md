# The length class

Use it like this:

```java
Length.of(10,M)
Length.of(156,CM)
Length.of(10,MM)
```
or import static method ```lengthOf``` and use it like this:
```java
lengthOf(10,M)
lengthOf(156,CM)
lengthOf(10,MM)
```
or simply use the constructor
```java
new Length(10,M)
new Length(156,CM)
new Length(10,MM)
```
